02

Framework CO-STAR

El framework que transforma prompts vagos en instrucciones precisas. 6 dimensiones: Contexto, Objetivo, Style, Tono, Audiencia, Respuesta.

C Contexto — "Que necesita saber la IA?" Ejemplo: "Soy estudiante de 4o ano, tesis de biblioteca comunitaria en Chihuahua, terreno 800m2"
O Objetivo — "Que resultado necesito?" Ejemplo: "Desarrollar un programa arquitectonico con areas en m2"
S Style — "Como debe sonar?" Ejemplo: "Como un documento profesional de programacion arquitectonica"
T Tono — "Que emocion transmite?" Ejemplo: "Formal pero empatico con la comunidad"
A Audiencia — "Para quien es?" Ejemplo: "Mi comite de tesis y stakeholders comunitarios"
R Respuesta — "Como lo quiero?" Ejemplo: "Tabla con m2, lista de principios, constraints en bullets"
03

Prompts de Imagen — 7 Capas

La formula para generar renders arquitectonicos: tipo de toma + sujeto + materiales + iluminacion + contexto + atmosfera + parametros.

Formula: [Tipo de toma] + [Sujeto + Estilo] + [Materiales] + [Iluminacion] + [Contexto] + [Atmosfera] + [Parametros]

Tip clave: Agrega lente para mas realismo: "shot on Hasselblad, 35mm lens" = estandar de fotografia arquitectonica. "24mm wide-angle" = interiores expansivos. "tilt-shift" = efecto maqueta para aereas.

04

Video Pipeline

De render a video cinematografico: el pipeline completo Image-to-Video con Kling, Veo y CapCut. La regla de oro: motion intensity a 0.3-0.5.

Pipeline: Render → Video clip (Kling/Veo, 4-10 seg) → Edicion (CapCut) → Musica → Narrativa

Regla critica: La arquitectura es RIGIDA. Solo la camara, la luz y la atmosfera deben moverse. SIEMPRE baja motion intensity a 0.3-0.5 o tus edificios se van a deformar.

05

Presentaciones con IA

Estructura de pitch profesional: Hook, Contexto, Concepto, Diseno, Evidencia, Vision. La regla: tus slides son tu telon de fondo, no tu guion.

Estructura de 6 partes: Hook (15 seg) → Contexto (30-45 seg) → Concepto (30-45 seg) → Diseno (60 seg) → Evidencia (30 seg) → Vision (15-30 seg)

Regla de oro: Lo que esta en pantalla NO es lo que dices. Slides = tu telon. Tu voz = la narrativa, la emocion, el argumento.

06

Colaboracion Humano-IA

El framework PAIR: Provide context, Ask specifically, Interrogate output, Refine iteratively. Tu criterio profesional es el firewall.

P — Proporcionar contexto: dale TODA la info del proyecto, sitio, clima, fase, restricciones.

A — Preguntar especificamente: usa CO-STAR, se preciso.

I — Interrogar el resultado: lee criticamente, busca errores, sesgos, alucinaciones.

R — Refinar iterativamente: itera 3-5 veces. NUNCA te quedes con el primer borrador.

Glosario

Los terminos que vas a dominar

Estos son los conceptos clave. Cuando termines el curso, vas a hablar de IA con confianza.

LLM

Large Language Model

Red neuronal entrenada con textos masivos para entender y generar lenguaje humano. Es el cerebro detras de ChatGPT y Claude.

Analogia: Imagina un arquitecto que ha leido TODOS los libros de arquitectura del mundo. Puede hablar de cualquier tema, pero nunca ha construido un edificio real.

Token

Token

Unidad minima de texto que procesa la IA — aproximadamente 3-4 caracteres. El espanol usa ~60% mas tokens que el ingles.

Analogia: Como los ladrillos de un muro. La IA no lee palabras completas — lee pedacitos y los ensambla.

Context Window

Ventana de Contexto

Cantidad maxima de tokens que la IA puede "ver" a la vez. Claude Opus 4.6: 1,000,000 tokens (~1,500 paginas).

Analogia: El tamano del escritorio de trabajo. Mas grande = mas documentos puedes tener abiertos al mismo tiempo.

Prompt

Prompt

Todo lo que escribes y envias a la IA — tu instruccion completa. La calidad de tu prompt determina la calidad de la respuesta.

Analogia: Como el brief que le das a un cliente. Si el brief es vago, el resultado es vago. Si es preciso, el resultado es preciso.

Alucinacion

Hallucination

Cuando la IA dice algo falso con total confianza. NO es un bug — es una caracteristica de como funcionan los LLMs. Modelos top: ~90-95% precision general.

Analogia: Como cuando un companero te da un dato con total seguridad pero se lo invento. Suena real, pero hay que verificar.

Difusion

Diffusion

Tecnica de generacion de imagenes: de ruido puro a imagen coherente. Es como la IA "ve" y "dibuja".

Analogia: Imagina que empiezas con una hoja llena de estatica de television y poco a poco, paso a paso, va apareciendo una foto. Asi funciona.

Temperature

Temperatura

Control de aleatoriedad de la IA. 0 = predecible y seguro. 1 = creativo e inesperado. 2 = caotico.

Analogia: Como el volumen de "locura creativa". Bajo para calculos y datos, alto para brainstorming y exploracion.

CO-STAR

Framework

El framework para escribir prompts efectivos: Contexto, Objetivo, Style, Tono, Audiencia, Respuesta. Tu herramienta #1 del curso.

Analogia: Como llenar los datos de un proyecto antes de disenar: sitio, programa, cliente, restricciones, formato de entrega.

Vibe Coding

Vibe Coding

Describes lo que quieres en lenguaje natural y la IA escribe el codigo. No necesitas saber programar — solo saber pedir.

Analogia: Como explicarle a un maestro de obra lo que quieres y el lo construye. Tu pones la vision, la IA pone la ejecucion tecnica.

Agent

Agente

IA que toma acciones autonomas, no solo responde preguntas. Puede crear archivos, ejecutar codigo, navegar la web.

Analogia: La diferencia entre un asistente que contesta el telefono y uno que ademas agenda citas, manda correos y organiza tu oficina.

MCP

Model Context Protocol

Estandar para conectar IA a herramientas externas. Permite que la IA use bases de datos, APIs y servicios directamente.

Analogia: Como el USB para la IA. Un conector universal que le permite hablar con cualquier herramienta.
